# simple-neuron

Training of a neuron

## Comand

```sh
python3 main.py
```

Wait a minute

You will get a neuron with:

```python
w[0] = 3.3086
w[1] = 2.1264
b = -39.6088
```

And you can put your data.

Example:
```python
# input
x1=4
x2=8
# output
y=0

# input
x1=8
x2=9
# output
y=1
```

### Alternative command

You can run an alternative method, that use the middle point of the dataset

```sh
python3 main.py middle
```
